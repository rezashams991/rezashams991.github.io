Recent Projects

OpenAppointments is a desktop appointment and scheduling application that I have been actively developing in Python. The project is built with a fully graphical user interface and follows a modular Object-Oriented Programming (OOP) architecture, making the codebase clean, maintainable, and easy to extend. The application supports multiple languages, light and dark themes, batch operations, and importing or exporting data in various file formats. It also includes folder-based organization, an archive system for completed records, and the ability to generate printable appointment cards, making it suitable for managing appointments in an organized and efficient way. This project serves as both a practical scheduling solution and an opportunity to apply software engineering principles such as modular design, reusable components, and scalable architecture. Once development is complete, the project will be released as open source on my GitHub profile.
